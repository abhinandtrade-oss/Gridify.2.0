# Online PDF Editor

A modern, clean, and professional web-based PDF editor.

## Features
- **PDF Rendering**: High-quality rendering using PDF.js.
- **Visual Editing**: Add text, draw shapes, and insert images with Fabric.js.
- **PDF Operations**: Rotate, delete, and reorder pages.
- **Export**: Generate high-quality PDFs with pdf-lib.
- **Analytics**: Usage tracking via Google Apps Script.

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- Libraries: PDF.js, Fabric.js, pdf-lib
- Backend: Node.js (Express)
